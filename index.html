Projects I have worked On

SAP Replatform

As a DevOps Consultant, I have worked on designing and setting up the infrastructure for SAP on Azure using Azure DevOps Pipeline, Terraform, Ansible and Powershell.

Cloud Migrations

Worked on multiple Cloud Advisory, Review and Migrations on both AWS and Azure. Used multiple tools to review and plan the migration of On-prem workloads on Cloud.

Enterprise Landing Zone

Designed and implemented the Enterprise Landing zone for both AWS and Azure for multiple customers. Followed the Cloud adoption framework to set up the multi-subscription and multi-tier architecture on Cloud.

Kubernetes Cluster Implementation

Designed and setup multiple Kubernetes Clusters on AWS, Azure and On-prem. Used the infrastructure as a code(Terraform) to implement the EKS and AKS for various customers.

DevOps Security

As a DevSecOps Consultant, helped many customers to move up the ladder in DevSecOps Maturity Model (DSOMM). Implemented SAST, DAST, IAST and pipeline security along with the threat modelling.

Kubernetes Security

Reviewd the security of Kubernetes setup for multiple customers and provided recommandations and remediations. Implented system and cluster hardening as well as Network and pod security polcies .
